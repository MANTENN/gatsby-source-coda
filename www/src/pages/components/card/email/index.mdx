Cards are used to callout certain features or actions Thumbtack is requesting of the recipient.

### Example

```email
{{> tp-card
    iconUrl="/img/icons/dollar_green@2x.png"
    title="When you’re done, rate this pro."
    text="Please respect professional boundaries. Inappropriate requests of any kind will not be tolerated and may be reported."
    linkText="Review this pro"
    linkHref="https://thumback.com"
}}
```

Not all options are required. In the following example there is no link, we’re simply letting the user know what will happen, not requesting an action.

```email
{{> tp-card
    iconUrl="/img/icons/charges@2x.png"
    title="What happens after 4 hours?"
    text="You’ll be needing to do someting that we’ll be sure to tell you about."
}}
```
