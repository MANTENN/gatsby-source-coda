## Avatar options

These are the available `Avatar` styles for use on light background colors.

### Pro

```email emailPartial=avatar
{{> avatar
    src="https://www.placecage.com/480/480"
    size="xlarge"
}}
```

### Customer

```email emailPartial=avatar
{{> avatar
    src="https://www.placecage.com/480/480"
    size="xlarge"
    type="customer"
}}
```

### Blank

These are the available `Avatar` styles for use on light background colors.

```email emailPartial=avatar
{{> avatar size="xlarge" }}
```
