## Button options

These are the two available `Button` styles for use on light background colors.

### Default

```email
{{> button
    href="https://thumbtack.com"
    text="Click me"
}}
```

### Secondary

```email
{{> button
    href="https://thumbtack.com"
    text="Click me"
    type="secondary"
}}
```

### Alignment

Buttons are left-aligned by default. They can be also be centered aligned with the `center` attribute.

```email
{{> button
    href="https://thumbtack.com"
    text="Centered button"
    align="center"
}}
```

### Full width

By default buttons will expand to the full-width of their container.

-   Buttons can be forced to expand on desktop with the `forceExpanded` attribute.
-   Buttons can be forced to to remain unexpanded on desktop and mobile with the `preventExpanded` attribute.

```email
{{> button
    href="https://thumbtack.com"
    text="Full width button"
    forceExpanded="true"
}}
```

```email
{{> button
    href="https://thumbtack.com"
    text="Full width button"
    preventExpanded="true"
}}
```
