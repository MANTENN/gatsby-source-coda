<!-- ## `SmartImage`

The `SmartImage` component loads an image from Thumbtack's Image Service. By default, it lazy-loads the image and [uses responsive images](https://developer.mozilla.org/en-US/docs/Learn/HTML/Multimedia_and_embedding/Responsive_images) to load the best size.

### With just an Image Service `id`

```jsx
<SmartImage id="350357374752325652" />
```

### With a custom `objectFit`

```jsx
<SmartImage id="350357374752325652" objectFit={{ height: 200, style: 'cover' }} />
```

### With a supported `aspectRatio`

```jsx
<SmartImage id="350357374752325652" aspectRatio="1:1" />
```

### With lazy-loading turned off

```jsx
<SmartImage id="356206765797793818" lazyLoad={null} />
```

### With a `png`

```jsx
<SmartImage id="356301205135450141" format="png" />
``` -->
