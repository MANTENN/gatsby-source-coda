Loading animations let the user know that content is on the way. The more consistent we are with the experience the more trust that the requested content will load momentarily. Because we’re asking the user to wait, we want take steps to limit this delay and, where possible, preview the content that is coming.

## Loading types

There are three primary loading types that are recommended for use in Thumbtack products:

1. A small loading dot animation.
2. Skeleton loaders.
3. The operating system loader for native apps.

### 1. Loading dots

This is our standard loading animation. This should be used inside buttons following an action, with a transparent white overlay to prevent additional actions, or along side text denoting a process is taking place.
