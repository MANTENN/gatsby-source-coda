Loading animations let the user know that content is on the way. The more consistent we are with the experience the more trust we’ll build with our users that they’ll get what they requested. Because we’re asking the user to wait, we want take steps to limit this delay and, where possible, preview the content that is coming.

## Loading types

There are three primary loading types that are recommended for use in Thumbtack products:

1. A small loading dot animation.
2. Skeleton loaders.
3. For native apps only, the operating system loader.

Deciding which one to use will require some judgement.

### 1. Loading dots

This is our standard loading animation. It should be used when if we need to keep the user on the page or when smaller pieces of an existing layout is being generated.
